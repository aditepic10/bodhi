# ADR-007: Bi-Temporal Facts

## Status
Accepted

## Context
Bodhi must answer questions about current state and change over time.

## Decision
Store facts with provenance, status, confidence, and validity windows.

## Consequences
Fact history and supersession are explicit. The fact model is richer than a simple key-value store.

## Ground Truth
Graphiti, Zep-style memory systems, temporal fact modeling.
