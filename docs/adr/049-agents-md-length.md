# ADR-049: AGENTS.md Length Discipline

## Status
Accepted

## Context
Verbose agent instruction files dilute the few rules that actually matter.

## Decision
Keep `AGENTS.md` short and limited to non-discoverable information.

## Consequences
Agents get higher-signal instructions. Context lives in code and docs instead of bloated onboarding files.

## Ground Truth
HumanLayer research on agent instruction effectiveness.
