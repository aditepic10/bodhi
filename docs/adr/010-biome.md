# ADR-010: Biome

## Status
Accepted

## Context
Bodhi needs fast formatting and linting with low config overhead.

## Decision
Use Biome instead of a separate ESLint + Prettier stack.

## Consequences
Tooling remains fast and unified. The project accepts Biome's opinions and upgrade cadence.

## Ground Truth
OpenCode, Biome's consolidated formatter/linter model.
